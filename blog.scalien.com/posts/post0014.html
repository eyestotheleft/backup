Keyspace 1.6.1: Java, PHP and Ruby libraries keyspace-1.6.1-java-php-ruby-libraries 2010-04-01 Marton Trencseni Great news, we're releasing Keyspace 1.6.1. The server binary has not changed other than the version number bump, but the package now includes client libraries for Java, PHP and Ruby. In specifying the client interfaces we tried to adhere to the local customs of the language. If you have any comments, suggestions or bug reports, please do so on the Keyspace Google Groups. Installation and programming are covered in the appropriate parts of the documentation:
  • Python API
  • Java API
  • PHP API
  • Ruby API
Coming up: support for .NET, possibly a Windows friendly C++ version of the client libraries.
