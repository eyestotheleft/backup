Keyspace 1.0.1 Hotfix Release keyspace-1.0.1-hotfix-release 2009/07/17 Marton Trencseni We are releasing 1.0.1 today, which fixes some minor issues users experienced.
  1. Keyspace would not start without a config file in default mode. Resolution: fixed.
  2. Keyspace printed a non-informative error message when a directory 'keyspace' was present and it tried to create the database file 'keyspace'. Resolution: More informative error message.
  3. Some Linux optimizations were accidentally removed in the 1.0 release. Resolution: added to 1.0.1
